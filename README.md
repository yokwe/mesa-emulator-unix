# mesa-emulator-unix
Xerox Mesa Emulator

Use standard c++ library for emulator and Tcl/Tk for GUI and control.
Emulator can boot GVWin with GUI. It takes abou 7 seconds from MP 0900 to 8000.

Emulator become stable. It can run more than 24 hours without issues.

Current developmen is on macOS.

Still in development. Don't expect it will work as you expecting.
